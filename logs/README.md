# 実行ログ

本ディレクトリは、WMSサンプルシステムの実行ログやデモ実施記録を保存するための場所です。

## 命名規則

ファイル名は以下の形式とします。

`{YYYY-MM-DD}_{NN}_{Title}.md`

- `YYYY-MM-DD`: ログ出力日 (例: 2026-03-16)
- `NN`: その日の連番 (01, 02, ...)
- `Title`: 実施内容を表す簡潔なタイトル (英語推奨、例: InitialDemo, DataLoadTest)

例: `2026-03-16_01_InitialConsoleDemo.md`

## 運用ルール

- 重要なデモや検証を行った際は、必ずログを残してください。
- ログには実行したコマンドとその結果、および考察を含めてください。
- テンプレート (`TEMPLATE.md`) をコピーして使用してください。
