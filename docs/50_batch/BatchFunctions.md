# バッチ一覧

## 概要

本WMSサンプルに含まれるバッチ処理の仕様です。各バッチの実行契機、入出力、更新対象、異常時対応を定義します。

---

## 1. 日次在庫集計

| 項目 | 内容 |
|------|------|
| バッチID | B001 |
| 名称 | 日次在庫集計 |
| 実行契機 | 日次（例: 23:00） |
| 入力 | Stock, Location, Item |
| 出力 | 日次在庫集計表（倉庫・商品別サマリ） |
| 更新 | なし（集計結果はファイルまたは一時テーブルに出力） |
| 異常時対応 | 集計対象データなしの場合は空ファイル出力。ログに記録 |

---

## 2. 出荷実績集計

| 項目 | 内容 |
|------|------|
| バッチID | B002 |
| 名称 | 出荷実績集計 |
| 実行契機 | 日次（例: 24:00） |
| 入力 | Shipment, OutboundOrder, Customer, Item |
| 出力 | 出荷実績集計表（日別・得意先別・商品別） |
| 更新 | なし |
| 異常時対応 | 集計対象なしの場合は空ファイル。異常終了時はリトライ対象 |

---

## 3. 棚卸差異出力

| 項目 | 内容 |
|------|------|
| バッチID | B003 |
| 名称 | 棚卸差異出力 |
| 実行契機 | 棚卸完了後、または日次（指定日付の棚卸データを対象） |
| 入力 | InventoryCount, Stock, Item, Location |
| 出力 | 棚卸差異一覧（帳簿在庫と実棚卸の差異） |
| 更新 | なし |
| 異常時対応 | 棚卸データなしの場合は終了コード1で終了。ログに記録 |

---

## 4. ログ整理

| 項目 | 内容 |
|------|------|
| バッチID | B004 |
| 名称 | ログ整理 |
| 実行契機 | 週次（例: 日曜 1:00） |
| 入力 | ログディレクトリ内のログファイル |
| 出力 | アーカイブファイル（圧縮）、または削除 |
| 更新 | ログファイルの移動・削除 |
| 異常時対応 | ファイルロック時はスキップして次へ。削除失敗はログに記録 |

---

## 実行順序の例

```mermaid
flowchart LR
    A[23:00 日次在庫集計] --> B[24:00 出荷実績集計]
    B --> C[棚卸差異出力: 棚卸日指定時]
    D[週次 ログ整理] -.-> A
```

日次バッチはオンライン処理が終了した後に実行することを想定しています。
